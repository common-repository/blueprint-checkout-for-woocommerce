Blueprint enables your customers to purchase with 1 message, via SMS or WhatsApp. No changing channel, no friction.

== Description ==

This plugin replaces your WooCommerce checkout with a hosted solution by Blueprint, in order to use this plugin you will need to sign up to Blueprint.

= Communicate 1:1 with your customers. =

Connected communication across the entire customer journey. Helping customers inform product purchases, optimise abandoned carts, and build relationships post-purchase.

= A simple 1-page checkout integration. =

From your customers first order and beyond, see the whole journey live in your dashboard.


Use our WooCommerce WordPress plugin to connect your store to Blueprint and start harnessing our technology today.

Please note an API key is required for this plugin to work, which can be retrieved upon completion of registration at https://blueprint.store/


== Installation ==

1. Install Blueprint checkout for WooCommerce plugin from WordPress Plugin Store
2. Activate Plugin
3. Obtain API KEY from Blueprint admin panel
4. Go to WordPress Admin > WooCommerce > Settings > Payments
5. Disable all payment gateways
6. Enable Blueprint checkout for WooCommerce
7. Click Set Up / Manage
8. Disable Test Mode and add your API KEY to the Live Api Key field
9. Save changes
